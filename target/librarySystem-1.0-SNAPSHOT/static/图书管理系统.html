Library Manager System项目简介项目构建

 

Library Manager System

 

这是一个简单的SSM整合项目，实现的功能还不是很多，由于一个人构建整个项目，所以需要浪费一些时间去网上找一些如 html 页面、前端插件库、图片等资源。

总之，这个项目非常的简单，大家可以继续去完善它，源文件下载地址已挂在最后。

 

项目简介

 

开发环境

  • IDEA
  • MySQL 5.7.19
  • Tomcat 9
  • Windows

技术点

  • 基础框架：SSM（Spring+SpringMVC+MyBatis）
  • 关系型数据库：MySQL
  • 项目依赖管理工具：Maven
  • Web 框架：BootStrap
  • Web 技术：HTML、CSS、JavaScript、JSP、jQuery、AJAX、JSON
  • 请求URI采用 RESTFUL 风格1URI

功能点

  1. 前端分页
  2. 前端通知框：使用了 jQuery 的 Toast 插件
  3. 前端弹窗：使用了 SweetAlert 插件
  4. AJAX 局部刷新
  5. CRUD2
